# texture.jpg — 캔버스 질감 이미지

로딩 화면의 "거장의 붓 터치" 캔버스에 오버레이되는 질감 이미지입니다.

## 필요한 이미지

- **종류**: 거친 유화 캔버스 질감 (Canvas / Linen texture)
- **톤**: 그레이스케일 또는 약한 웜톤 (너무 진하면 로딩 UI를 가림)
- **해상도**: 512×512 ~ 1024×1024 (타일링 가능하면 좋음)
- **파일명**: `texture.jpg`
- **위치**: `public/texture.jpg`

## 추천 소스

- [Unsplash](https://unsplash.com/s/photos/canvas-texture) — "canvas texture" 검색
- [Pexels](https://www.pexels.com/search/canvas%20texture/) — 무료 텍스처
- [Poly Haven](https://polyhaven.com/textures) — CC0 텍스처

## 예시 키워드

`canvas weave`, `linen texture`, `oil painting canvas`, `rough paper texture`

파일을 추가하면 로딩 캔버스에 CSS 패턴 대신 이 이미지가 오버레이됩니다.
