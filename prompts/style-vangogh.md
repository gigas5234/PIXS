# style-vangogh

## Concept
- 강렬한 임파스토
- 소용돌이치는 질감
- 보색 대비의 과감한 색채

## Key
- #SwirlingCanvas
- #ThickOilPaint
- #VibrantYellow
- #PostImpressionism

## Prompt
Generate a dynamic post-impressionist pet portrait inspired by Van Gogh. Emphasize thick impasto brushwork, swirling directional strokes, and expressive color contrast. Use vivid yellows against cooler complementary tones to create emotional intensity while preserving recognizable facial structure and character.
